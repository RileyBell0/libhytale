the structure is

```json
{
    "BlockType" {
        ...

        "Interactions": {
            "<enum key of InteractionTypes type>": {
                "Interactions": [
                    {
                        "Type": "<id of your interaction>"
                    }
                ]
            }
        }
    }
}
```
e.g.

```json
{
    "BlockType" {
        "Interactions": {
            "Use": {
                "Interactions": [
                    {
                        "Type": "dev.twunk.interactions.OpenChestInteraction"
                    }
                ]
            },
            "Primary":  {
                "Interactions": [
                    {
                        "Type": "Break_Block"
                    }
                ]
            }
        }
    }
}
```
